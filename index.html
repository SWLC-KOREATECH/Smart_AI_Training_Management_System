[Part A] 자동 알림 봇

1. 시트 입력 규칙 및 검증

각 열(Column)은 정해진 규칙대로 입력해야 봇이 인식합니다. 규칙을 어길 경우 I열(상태)에 에러가 표시됩니다.

열 구분 입력 내용 주의 사항 / 검증 로직
A~C열
(날짜/시간)		 2025-05-01 / 14:00
  • 날짜/시간 형식이 아니면 입력 거부됨
  • 종료 시간이 시작 시간보다 빠르면 에러 발생
D열
(알림 키워드)		 한기대 / 일학습
  • 가장 중요! 알림을 보낼 채널을 결정하는 키워드
  • 이 키워드에 따라 봇이 #공지-한기대반 또는 #공지-일학습반을 구분함
  • 오타가 있을 경우 "알 수 없는 키워드" 에러 발생
E열
(일정 제목)		 자바 프로그래밍 기초
  • 캘린더와 슬랙 알림 메시지에 표시될 실제 일정 제목
  • 말머리를 제외한 순수 제목만 입력하세요.
F~G열
(장소, 추가 안내사항)		 A관 305호, 노트북 지참
  • 여기에 적은 내용은 '시작 10분 전 알림'에만 포함됨
  • 공란 시 알림에서 내용 생략
H열
(작업선택)		 등록 / 삭제
  • 드롭다운 메뉴로 선택
  • 선택 후 반드시 상단 [작업 실행하기] 버튼을 눌러야 반영됨
I~J열
(시스템)		 자동 입력됨 봇이 처리 결과(등록완료/에러)와 고유 ID를 기록하는 곳입니다.
사용자는 수정하지 마세요. (J열은 숨김 처리 추천)

2. 알림 발송 원리

D열에 입력한 [말머리]에 따라 알림이 가는 방이 달라집니다. (Apps Script 설정에서 변경 가능)

[한기대] #공지-한기대반
[일학습] #공지-일학습반
[공통] 모든 채널 동시 발송

3. 캘린더 등록 위치

사용자가 [작업 실행하기]를 누르면, 봇은 내 구글 계정에 있는 특정 이름의 캘린더를 찾아 일정을 넣습니다.

주의 사항
기본 설정은 '출결'이라는 이름의 캘린더를 찾습니다.
만약 내 구글 캘린더에 '출결'이라는 캘린더가 없으면 봇이 멈추고 에러 메시지를 띄웁니다. (Step 0 참조)

[Part B] AI 지능형 튜터

슬랙 채팅창에서 명령어(/)를 입력하면 AI가 즉시 답변합니다.

/일지 [키워드] 핵심 기능

훈련내용(키워드)을 입력하면, 학습활동서 예시를 작성해줍니다.

사용 예: /일지 에스프레소 추출 실습, 원두 분쇄도 조절, 우유 스팀 온도 맞추기
/안전 [작업명 또는 장비명]

해당 작업이나 장비 사용 전 반드시 확인해야 할 안전 수칙을 알려줍니다.

사용 예: /안전 용접기 사용
/질문 [궁금한 용어]

낯선 기술 용어나 개념을 쉽게 설명해줍니다.

사용 예: /질문 인터록 회로가 뭐야?
/도움말

Part A. 인프라 설정
Step 1-1

구글 캘린더 생성

일정이 등록될 전용 캘린더를 먼저 만들어야 합니다.

  1. 구글 캘린더(클릭)에 접속합니다.
  2. 왼쪽 메뉴의 '다른 캘린더' 옆 [+] 버튼 클릭 → [새 캘린더 만들기] 선택
  3. 이름에 출결 (또는 원하는 이름) 입력 후 [캘린더 만들기] 클릭
  4. *중요: 이 캘린더의 이름을 정확히 기억해두세요. (나중에 설정에 입력해야 함)
Step 1-2

Slack 워크스페이스 및 채널 준비

알림을 받을 공간(워크스페이스)과 방(채널)을 먼저 만들어야 합니다.

  1. Slack(클릭)에 접속하여 회원가입 또는 구글 계정으로 로그인 후 [워크스페이스 생성]을 진행합니다. (이미 있다면 건너뛰기)
  2. 훈련 과정별로 알림을 받을 채널을 생성합니다.
    • 예시 1: #공지-한기대반 (한기대 훈련생용)
    • 예시 2: #공지-일학습반 (일학습 훈련생용)
    • 예시 3: #관리자-알림 (담당자 확인용)
  3. *연계 포인트: 여기서 만든 채널 각각에 대해 고유한 주소(Webhook URL)를 발급받아 코드에 연결하게 됩니다.
Step 1-3

Groq 무료 API Key 발급

AI 튜터 기능을 무료로 사용하기 위해 Groq API 키를 발급받아야 합니다.

  1. Groq Console(클릭)에 접속하여 회원가입 또는 구글 계정으로 로그인합니다.
  2. 화면 중앙(또는 우측 상단)의 [Create API Key] 버튼을 클릭합니다.
  3. Name에 AI_Tutor 등 원하는 이름을 입력하고 [Submit]을 누릅니다.
  4. 생성된 키(gsk_...로 시작)를 복사하여 메모장에 잘 저장해둡니다.
    중요: 창을 닫으면 키를 다시 볼 수 없으니 생성 즉시 복사해두세요!
Step 2

Slack 앱 간편 생성 (Manifest 활용)

복잡한 설정(권한, 명령어 등)을 코드 붙여넣기 한 번으로 끝냅니다.

  1. Slack API 사이트 접속 → 우측 상단 [Create New App] → [From a manifest] 선택.
  2. 설치할 워크스페이스 선택 → [Next] 클릭.
  3. 형식 선택에서 [JSON] 탭 클릭 → 기존 내용을 지우고 아래 코드를 복사해 붙여넣기 → [Next] → [Create].
    JSON Code

    * 위 JSON 코드가 봇 권한(Scopes)을 자동으로 생성해줍니다.

  4. [필수 설정 1] Slack 앱 이름 변경:
    앱이 생성되면 왼쪽 메뉴 [App Home] 이동 → 화면 중앙 [App Display Name] Edit 버튼 클릭 → [Display Name (Bot Name)]에 기존 이름 삭제 후 앱 이름 입력(훈련일정 알리미) 후 저장.
  5. [필수 설정 2] 알림 주소(Webhook) 발급:
    왼쪽 메뉴 [Incoming Webhooks] 이동 → 하단 [Add New Webhook to Workspace] 클릭 → Step 1-2에서 만든 키워드 채널(#관리자-알림, #공지-한기대반 등) 선택 후 [Allow].
    * 생성된 Webhook URL을 복사해 메모장에 저장 (채널별 반복)
Step 3

훈련 일정표 양식 복사

자동화 코드가 이미 심어져 있는 엑셀 양식을 내 구글 드라이브로 가져옵니다.

양식 복사하기 (클릭)

* 클릭 후 [사본 만들기]를 누르면 내 구글 드라이브에 저장됩니다.
* 구글 로그인 후 사용가능

Step 4

설정 변경 및 트리거 연결

복사된 시트에 내 정보(캘린더, 슬랙 주소)를 입력하는 마지막 단계입니다.

  1. 복사된 스프레드시트 상단 메뉴 [확장 프로그램] → [Apps Script] 클릭
  2. 코드 편집기가 열리면 맨 윗부분의 설정을 수정합니다.
    • [기본 설정 1] TARGET_CALENDAR_NAME: Step 0에서 만든 캘린더 이름 ('출결')
    • [기본 설정 2] ADMIN_WEBHOOK: Step 2. [필수 설정 2]에서 확보한 '#관리자-알림' Webhook URL
    • [기본 설정 3] CHANNEL_MAP: Step 2. [필수 설정 2]에서 확보한 '#공지-일학습반, #공지-한기대반' Webhook URL을 각각 입력
  3. 저장(Ctrl+S) 후, 왼쪽 메뉴의 시계 아이콘(트리거)을 클릭합니다.
  4. 우측 하단 [+ 트리거 추가] 버튼 클릭
  5. 실행할 함수: checkRealTimeAlerts / 이벤트 소스: 시간 기반 / 유형: 분 단위 / 간격: 1분마다
  6. [저장] 클릭 (최초 1회 권한 승인 필요)
Part B. Groq AI 튜터 엔진 설정 (Apps Script)
Step 1

Google Apps Script 배포 (서버 열기)

작성하신 코드를 슬랙이 접속할 수 있는 '웹 사이트 주소'로 만드는 과정입니다.

  1. Apps Script 우측 상단 [배포] → [배포 관리] 클릭 후 유형선택 톱니바퀴 → 웹 앱 클릭.
  2. 최초 설정 (⭐가장 중요): 추후 수정 시에는 하단 💡 유지보수 핵심 요약 참고.
    • 버전 (Version): 반드시 [새 버전 (New version)] 선택. (수정 시 필수)
    • 다음 사용자로서 실행: [나 (Me)]
    • 액세스 권한: [모든 사용자 (Anyone)] (슬랙 접속 허용)
  3. [배포] 버튼 클릭 후 생성된 '웹 앱 URL'을 복사해 둡니다.
Step 2

Slack 앱 간편 생성 (Manifest 활용)

복잡한 설정(권한, 명령어 등)을 코드 붙여넣기 한 번으로 끝냅니다.

  1. Slack API 사이트 접속 → 우측 상단 [Create New App] → [From a manifest] 선택.
  2. 설치할 워크스페이스 선택 → [Next] 클릭.
  3. 형식 선택에서 [JSON] 탭 클릭 → 기존 내용을 지우고 아래 코드를 복사해 붙여넣기 → [Next] → [Create].
    JSON Code

    * 위 JSON 코드가 봇 권한(Scopes)과 명령어 4개를 자동으로 생성해줍니다.

  4. [필수 설정 1] Slack 앱 이름 변경:
    앱이 생성되면 왼쪽 메뉴 [App Home] 이동 → 화면 중앙 [App Display Name] Edit 버튼 클릭 → [Display Name (Bot Name)]에 기존 이름 삭제 후 앱 이름 입력(훈련일정 알리미) 후 저장.
  5. [필수 설정 2] 알림 주소(Webhook) 발급:
    왼쪽 메뉴 [Incoming Webhooks] 이동 → 하단 [Add New Webhook to Workspace] 클릭 → Step 1-2에서 만든 키워드 채널(#공지-한기대반 등) 선택 후 [Allow].
    * 생성된 Webhook URL을 복사해 메모장에 저장 (채널별 반복)
  6. [필수 설정 3] Request URL 연결:
    왼쪽 메뉴 [Slash Commands] 이동 → 아무 명령어의 연필 아이콘(Edit) 클릭.
    Request URL 칸에 있는 임시 주소(example.com)를 지우고, Apps Script 배포 주소(Web App URL)로 변경 후 저장.
    (※ 이 작업은 Part B. Step 1에서 웹 앱 URL이 나온 뒤에 진행하세요!)
Step 3

워크스페이스 설치 (연결 마무리)

설정을 마친 앱을 실제 슬랙 방으로 초대하는 과정입니다.

  1. 왼쪽 메뉴 [Install App] → [Install to Workspace] (또는 Reinstall) 클릭.
  2. 워크스페이스 및 채널 선택 (⭐중요): 워크스페이스 선택 드롭다운에서 생성한 워크스페이스 선택 → 채널 선택 드롭다운에서 다이렉트 메시지 중 [AI 튜터] 채널을 선택하세요.
  3. [허용 (Allow)] 클릭.
Step 4

슬랙에서 확인 및 실행

  1. 앱 찾기: 슬랙 사이드바 하단 [앱] 아래 [+ 앱 추가] 클릭 → "AI 튜터"(또는 만든 이름) 검색 후 추가.
  2. 테스트: 채팅창에 /도움말을 입력하여 봇이 정상적으로 응답하는지 확인합니다.
