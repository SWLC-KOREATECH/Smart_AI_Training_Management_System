🚀 초기 설치 가이드

시스템 구축을 위해 순서대로 따라해 주세요.

Step 0. 구글 캘린더 생성

일정이 등록될 전용 캘린더를 먼저 만들어야 합니다.

  1. 구글 캘린더(클릭)에 접속합니다.
  2. 왼쪽 메뉴의 '다른 캘린더' 옆 [+] 버튼 클릭 → [새 캘린더 만들기] 선택
  3. 이름에 출결 (또는 원하는 이름) 입력 후 [캘린더 만들기] 클릭
  4. *중요: 이 캘린더의 이름을 정확히 기억해두세요. (나중에 설정에 입력해야 함)

Step 1. 슬랙 워크스페이스 및 채널 준비

알림을 받을 공간(워크스페이스)과 방(채널)을 먼저 만들어야 합니다.

  1. 슬랙(Slack)에 접속하여 회원가입 후 [워크스페이스 생성]을 진행합니다. (이미 있다면 건너뛰기)
  2. 훈련 과정별로 알림을 받을 채널을 생성합니다.
    • 예시 1: #공지-한기대반 (한기대 훈련생용)
    • 예시 2: #공지-일학습반 (일학습 훈련생용)
    • 예시 3: #관리자-알림 (담당자 확인용)
  3. *연계 포인트: 여기서 만든 채널 각각에 대해 고유한 주소(Webhook URL)를 발급받아 코드에 연결하게 됩니다.

Step 2. 슬랙 앱(Bot) 생성 및 연결

만들어둔 채널에 메시지를 보낼 '메신저 봇'을 만드는 과정입니다.

  1. Slack API 사이트(클릭)에 접속합니다. (※ 회원가입한 슬랙 계정으로 로그인이 필요합니다)
  2. 우측 상단의 [Create New App] → [From scratch] 선택.
  3. App Name(예: 훈련출결 알리미) 입력 후 워크스페이스를 선택하고 [Create App] 클릭.
  4. 왼쪽 메뉴 [Incoming Webhooks] → 스위치를 [On]으로 켜기.
  5. 화면 하단 [Add New Webhook to Workspace] 버튼 클릭.
  6. Step 1에서 만든 채널 중 하나(예: #공지-한기대반)를 선택하고 [Allow] 클릭.
  7. 생성된 Webhook URL (https://hooks.slack.com/...)을 복사하여 메모장에 '한기대반 주소'라고 적어둡니다.
  8. *반복: 다른 채널(일학습반, 관리자용)도 위 과정을 반복하여 각각의 주소를 따로 챙겨둡니다.

Step 3. 훈련 일정표 양식 복사

자동화 코드가 이미 심어져 있는 엑셀 양식을 내 구글 드라이브로 가져옵니다.

양식 복사하기 (클릭)

* 클릭 후 [사본 만들기]를 누르면 내 구글 드라이브에 저장됩니다.
* 구글 로그인 후 사용가능

Step 4. 설정 변경 및 트리거 연결

복사된 시트에 내 정보(캘린더, 슬랙 주소)를 입력하는 마지막 단계입니다.

  1. 복사된 스프레드시트 상단 메뉴 [확장 프로그램] → [Apps Script] 클릭
  2. 코드 편집기가 열리면 맨 윗부분의 설정을 수정합니다.
    • TARGET_CALENDAR_NAME: Step 0에서 만든 캘린더 이름 ('출결')
    • ADMIN_WEBHOOK: Step 2에서 확보한 '관리자용' 웹훅 주소
    • CHANNEL_MAP: Step 2에서 확보한 '채널별' 웹훅 주소를 각각 입력
  3. 저장(Ctrl+S) 후, 왼쪽 메뉴의 시계 아이콘(트리거)을 클릭합니다.
  4. 우측 하단 [+ 트리거 추가] 버튼 클릭
  5. 실행할 함수: checkRealTimeAlerts / 이벤트 소스: 시간 기반 / 유형: 분 단위 / 간격: 1분마다
  6. [저장] 클릭 (최초 1회 권한 승인 필요)

상세 사용자 매뉴얼

복사된 스프레드시트의 올바른 사용법입니다.

1. 시트 입력 규칙 및 검증

각 열(Column)은 정해진 규칙대로 입력해야 봇이 인식합니다. 규칙을 어길 경우 H열(상태)에 에러가 표시됩니다.

열 구분 입력 내용 주의 사항 / 검증 로직
A~C열
(날짜/시간)		 2025-05-01 / 14:00
  • 날짜/시간 형식이 아니면 입력 거부됨
  • 종료 시간이 시작 시간보다 빠르면 에러 발생
D열
(일정 제목)		 [한기대] 자바 실습
  • 가장 중요! [키워드] 말머리 필수
  • 이 키워드를 보고 봇이 어떤 슬랙 채널로 보낼지 결정함
  • 말머리가 없으면 등록 시 "말머리 누락" 에러 표시됨
E~F열
(장소, 추가 안내사항)		 A관 305호, 노트북 지참
  • 여기에 적은 내용은 '시작 10분 전 알림'에만 포함됨
  • 공란 시 알림에서 내용 생략
  • 종료 알림에는 포함되지 않음
G열
(작업선택)		 등록 / 삭제
  • 드롭다운 메뉴로 선택
  • 선택 후 반드시 상단 [작업 실행하기] 버튼을 눌러야 반영됨
H~I열
(시스템)		 자동 입력됨 봇이 처리 결과(등록완료/에러)와 고유 ID를 기록하는 곳입니다.
사용자는 수정하지 마세요. (I열은 숨김 처리 추천)

2. 알림 발송 원리

D열에 입력한 [말머리]에 따라 알림이 가는 방이 달라집니다. (Apps Script 설정에서 변경 가능)

[한기대] #공지-한기대반
[일학습] #공지-일학습반
[공통] 모든 채널 동시 발송

3. 캘린더 등록 위치

사용자가 [작업 실행하기]를 누르면, 봇은 내 구글 계정에 있는 특정 이름의 캘린더를 찾아 일정을 넣습니다.

주의 사항
기본 설정은 '출결'이라는 이름의 캘린더를 찾습니다.
만약 내 구글 캘린더에 '출결'이라는 캘린더가 없으면 봇이 멈추고 에러 메시지를 띄웁니다. (Step 0 참조)
